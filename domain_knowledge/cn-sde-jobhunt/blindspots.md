# cn-sde-jobhunt — Blindspots (Layer 3)

Blindspot catalogue for `cn-sde-jobhunt`. Per `_schema.md` Layer 3
is **LLM-weak**: entries are hand-authored from real evidence
(一亩三分地 threads, Reddit cases, attorney case notes, returnee
narratives) — not generated. The structure below organizes entries
by anchor framing (F1–F6 from `framings.md`); per-framing entries
require ≥ 5 each per `_schema.md`.

This file is structural-only: entries (statement / failure-mode /
recovery-move / trigger-situation) are voice-work, authored by the
human only.
