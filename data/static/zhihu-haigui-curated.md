# Zhihu 海归 / Returnee Voices — Curated Corpus

This file is the static corpus for source-views pointing at
`adapter: static_corpus` in the `china-returnee-voices` community
(Zhihu long-form posts, WeChat blog excerpts, and Chinese-language
podcast transcripts such as 硅谷101 and 软实力).

Zhihu's ToS allows individual citation but forbids systematic
scraping; WeChat and podcast platforms have similarly walled access.
This corpus is therefore manually curated rather than fetched.

## How to use

When the author encounters a Zhihu post, WeChat essay, or podcast
segment that exemplifies a load-bearing pattern for the CN-SDE-in-US
career decision (typically about return-to-China timing, US-CN
career-capital comparison, or the lived experience of having
crossed back), they:
1. Read the original.
2. Distill the load-bearing claims into 3–8 paragraphs here (their
   own paraphrase, not direct copy — respecting copyright).
3. Add a section header that includes: source platform, author
   handle (or anonymized), publication date, decision pattern, and
   a stable anchor for citation.

The Editor agent will cite chunks of this file as `[doc-X]` markers
just like any other source.
